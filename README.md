# Neural Architect Landing Page

Este proyecto es una página de aterrizaje para Neural Architect, que presenta un enfoque innovador en el procesamiento de señales biométricas y la escalabilidad en entornos de EdTech.

## Contenido del Proyecto

- **index.html**: Página de aterrizaje principal que contiene la estructura HTML y referencias a estilos y scripts.
- **README.md**: Documentación del proyecto, incluyendo información sobre la instalación, uso y contribuciones.
- **LICENSE**: Especifica la licencia bajo la cual se distribuye el proyecto.
- **.gitignore**: Lista de archivos y directorios que deben ser ignorados por Git.
- **package.json**: Configuración para npm, incluyendo dependencias y scripts.
- **src/css/styles.css**: Estilos CSS utilizados en la página de aterrizaje.
- **src/js/main.js**: Código JavaScript que maneja la interactividad de la página.
- **src/assets/fonts**: Fuentes utilizadas en el proyecto.
- **src/assets/svgs**: Archivos SVG utilizados en el proyecto.
- **.github/workflows/deploy.yml**: Flujo de trabajo para la implementación del proyecto en GitHub Actions.

## Instalación

1. Clona el repositorio:
   ```
   git clone https://github.com/tu_usuario/neural-architect-landing.git
   ```
2. Navega al directorio del proyecto:
   ```
   cd neural-architect-landing
   ```
3. Instala las dependencias:
   ```
   npm install
   ```

## Uso

Para iniciar el proyecto, abre el archivo `index.html` en tu navegador. También puedes configurar un servidor local para una mejor experiencia de desarrollo.
